Tech Specs

Hardware specifications of the RPi 4 and RPi Zero models

There have been fifteen (woah!) Raspberry Pi models released to date since the debut in 2012. The current list is: Raspberry Pi B, A, B+, and A+; Raspberry Pi 2 B; Raspberry Pi Zero and Zero W; Raspberry Pi 3 B, A+, and B+; Raspberry Pi 4 B 1 GiB, 2 GiB, 4 GiB, and 8 GiB; and Raspberry Pi 400. We have decided to showcase the current most powerful model, the Pi 4 B, as well as the smallest and cheapest model, the Pi Zero. Coming in at $35 and $10, respectively, these devices pack so much bang for your buck. The Pi Zero can also be upgraded to the Pi Zero W ('W' stands for wireless) for just an extra $5. Compare this to the $120+ you might have spent on a graphing calculator, and you can easily see the amazing value of these computers. Just plug in a monitor, keyboard, and mouse to the Pi 4 to get a very capable machine running a Linux-based, graphical operating system where you can browse the Internet, edit text documents, and program!

Raspberry Pi 4
  • Dimensions: 85mm x 56mm
  • CPU: Quad Core ARM-based @ 1.5 GHz
  • GPU: VideoCore VI Graphics
  • I/O: 2 USB 3.0, 2 USB 2.0, 2 Micro HDMI, Gigabit Ethernet, 40 GPIO Pins
  • RAM: 1 GB / 2GB / 4GB / 8GB
  • Storage: Micro SD Card
  • Additional Notes: Bluetooth 5.0, Wireless Networking @ 2.4 GHz and 5.0 GHz
Raspberry Pi Zero W
  • Dimensions: 65mm x 30mm
  • CPU: Single Core ARM-based @ 1 GHz
  • GPU: VideoCore IV Graphics
  • I/O: Mini HDMI, 2 Micro USB, 40 GPIO Pins
  • RAM: 512 MB
  • Storage: Micro SD Card
  • Additional Notes: Bluetooth 5.0, Wireless Networking @ 2.4 GHz
