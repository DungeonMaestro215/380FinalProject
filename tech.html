Tech Specs

Hardware specifications of the RPi 4 and RPi Zero models

There have been numerous Raspberry Pi models released sine the first RPi Model B in 2012.

  • Raspberry Pi B, A, B+, A+
  • Raspberry Pi 2 B
  • Raspberry Pi Zero and Zero W
  • Raspberry Pi 3 B, A+, and B+
  • Raspberry Pi 4 B, available in 1, 2, 4, and 8 GiB
  • Raspberry Pi 400
Raspberry Pi 4
  • CPU: Quad Core ARM-based @ 1.5 GHz
  • GPU: VideoCore VI Graphics
  • I/O: 2 USB 3.0, 2 USB 2.0, 2 Micro HDMI, Gigabit Ethernet, 40 GPIO Pins
  • RAM: 1 GB / 2GB / 4GB / 8GB
  • Storage: MicroSD Card
  • Additional Notes: Bluetooth 5.0, Wireless Networking 2.4 GHz and 5.0 GHz
Raspberry Pi Zero W
  • CPU: Single Core ARM-based @ 1 GHz
  • GPU:
  • I/O: Mini HDMI, 2 Micro USB
  • RAM: 512 MB
  • Storage: MicroSD Card
  • Additional Notes: Bluetooth 5.0, Wireless Networking
